# Skunch

##Simple Brunch.io Skeleton to build fast projects with  :
 - [Bootstrap 3](https://getbootstrap.com/docs/3.3/)
 - [Bootstrap select](http://silviomoreto.github.io/bootstrap-select/)
 - [Boostrap Datepicker](https://bootstrap-datepicker.readthedocs.io/en/latest/)
 - [Sass Brunch](https://github.com/brunch/sass-brunch)
 - [jQuery](http://jquery.com/)
 - [Static Handlebars Brunch](https://github.com/bloudermilk/static-handlebars-brunch)
